# The Other — Luxury Eyewear E-Commerce

A modern luxury eyewear e-commerce website built with React, Tailwind CSS, and Vite.

## Features

- **Product Catalog**: 39 sunglasses + 35 optical eyeglasses
- **Filter & Sort**: Filter by color and price, sort by newest or price
- **Shopping Cart**: Add to cart functionality
- **Responsive Design**: Mobile-first design with Tailwind CSS
- **Routing**: React Router for page navigation

## Pages

- `/` — Home
- `/sun` — Sun Collection (Sunglasses)
- `/optical` — Optical Collection (Eyeglasses)
- `/product/:id` — Product Detail
- `/about` — About
- `/contact` — Contact
- `/shipping` — Shipping Info
- `/returns` — Returns Info

## Tech Stack

- React 18
- Vite
- Tailwind CSS Browser
- React Router DOM
- Iconify Icons

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
